## Phase 1: 基礎骨架

> Status: completed

建立 PowerShell profile 入口與所有模組化載入腳本。

- `profile.ps1` 作為主入口，要求 PowerShell 7+，實作 `.env` 載入與 `profile.d/` 模組化載入
- `profile.d/` 包含 11 個模組：PATH helpers、aliases、PSReadLine、oh-my-posh prompt、額外模組、Node.js (fnm)、Python (pyenv-win)、Go、Rust、Bun、pnpm
- 所有模組實作條件載入，缺少工具時靜默跳過

完成日期: 2026-03-18

## Phase 2: 安裝腳本

> Status: completed

建立完整的模組化安裝腳本集。

- `setup.d/` 包含 16 個安裝腳本，涵蓋：scoop、Neovim、Node.js (fnm)、Python、Go、Rust、Bun、Nerd Fonts、oh-my-posh、GitHub CLI、kubectl、gcloud、ArgoCD CLI
- `setup_dotfiles.ps1` 處理配置檔 symlink/copy 到 Windows 標準路徑，支援權限自動偵測
- `setup_neovim.ps1` 將共用的 `nvim/init.lua` symlink 到 `%LOCALAPPDATA%\nvim\`
- `setup_claude.ps1` 處理 Claude Code 配置安裝
- 超出原始 SPEC 規劃，額外新增 gh、kubectl、gcloud、argocd 安裝腳本

完成日期: 2026-03-20

## Phase 3: 入口與文件

> Status: completed

建立統一安裝入口與使用文件。

- 根目錄 `setup.ps1` 提供互動式安裝選單，支援 `--all` 參數跳過選單
- 根目錄 `setup.bat` 作為薄 wrapper，雙擊即可執行，自動偵測 PowerShell 7+
- `script/windows/README.md` 包含需求、快速開始、目錄結構、元件對應表、Neovim/Windows Terminal/symlink 說明

完成日期: 2026-03-20
