- [x] 實作 `secrets-reload` helper function -- 清快取並重新解密
- [x] 實作 `secrets-init` helper function -- 初始化 age key pair + 更新 .sops.yaml
- [x] YAML 巢狀解析 -- 支援 `category.KEY` 轉為 `CATEGORY_KEY` 環境變數

## Phase 3: Setup 整合

- [x] `setup.sh` 加入 `install_sops.sh` 安裝項目
- [x] 建立 `script/utils/sec` CLI 工具 -- secret 管理快捷操作
  - [x] `sec init` -- 初始化 age key pair + .sops.yaml
  - [x] `sec edit` -- 編輯 secrets（vim yaml，存檔自動加密）
  - [x] `sec show` -- 顯示解密後的 secret（支援 bat 上色）
  - [x] `sec list` -- 只列 key 名稱（不顯示 value）
  - [x] `sec status` -- 檢查 age/sops/key/cache 狀態
  - [x] `sec push` -- 加密檔 commit + push 到 remote
  - [x] `sec export <file>` -- 匯出解密後的 secret（yaml 或 .env 格式）
  - [x] `sec import <file>` -- 從明文 yaml 匯入並加密
  - [x] `sec reload` -- 清快取，重新解密
- [ ] 首次 setup 引導 -- 偵測 age key 不存在時自動引導建立或匯入
