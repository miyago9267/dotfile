# 禁止 AI 署名

## 規則

1. git commit message 禁止加入 `Co-Authored-By` 或任何 AI 署名行（包括但不限於 Claude、GPT、Copilot 等）
2. AI 協作資訊只能記錄在文檔 summary（如 session summary、changelog），不得出現在 commit message 中
3. 不得在 commit message 尾部自動附加任何形式的 AI attribution trailer

## 反模式

- 不要在 commit message 加 `Co-Authored-By: Claude ...`
- 不要在 commit message 加 `Generated by ...` 或 `AI-assisted` 等字樣
