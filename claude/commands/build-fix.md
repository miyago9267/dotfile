# /build-fix

漸進式修復 TypeScript 或建置錯誤，一次修一個，確保不引入新問題。

## 流程

1. 執行建置（`npm run build` / `pnpm build` / `bun run build`）
2. 解析錯誤輸出，依檔案分組、依嚴重度排序
3. 每個錯誤：
   - 顯示錯誤上下文（前後 5 行）
   - 說明原因
   - 提出修復方案並套用
   - 重新建置驗證是否修好
4. 停止條件：
   - 修復引入新錯誤
   - 同一個錯誤修了 3 次還沒好
   - 使用者要求暫停
5. 最後回報：已修 / 未修 / 新增的錯誤數

每次只修一個錯誤，安全第一。
